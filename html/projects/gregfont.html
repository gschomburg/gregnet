Single
Stroke Fonts

GregFonts is a series of custom made single stroke fonts for use in vector design, laser cutting, and pen plotting.

While working on the Drawbot, I designed a set of monospace uppercase characters. Much like a plotter, the Drawbot uses vector data to plot its path, so using traditional fonts would would only create letter outlines. Originally I was planning on creating only the uppercase alphabet, but it was too much fun so I kept working on it. I ended up creating "Sutton" a complete single stroke "font" including a full set of letters, numbers and punctuation. Since then I've created another full font set, "Nassau", and have a few others in the works.

Using my single stroke fonts in other projects revealed some interesting properties I hadn't considered when building them. The letter forms can be distorted and warped independently of the stroke. The weight can be adjusted from bold to thin or dotted and dashed very easily. You can also apply brushes to the stroke in Illustrator to get more complex effects.

GregFont

The major disadvantage of using the single stroke fonts is laying out type. These letter forms cannot be represented in traditional font formats so there isn't a way to easily layout text. So I created one. GregFont allows you to paste in text, configure the font settings and then export as an svg. See GregFont in action here

Sutton

This is the first font I made. It was first used for Drawbot plotting, but was later refined and integrated into Combscript and also used for one of the Sketching in Hardware identities. It is a narrow minimal monospace font.

Nassau

While designing Sutton I kept wanting to add serifs to everything, but I couldn't. I created Nassau just so that I could use all of my serif ideas. Nassau is also not monospace, so it presented some new challenges in my tools for text layout.

Kosciuszko

Kos is very much a work in progress. My goal is to create a font that is intended to be warped and smoothed. I'm taking a lot of inspiration from some arts and crafts typography.
